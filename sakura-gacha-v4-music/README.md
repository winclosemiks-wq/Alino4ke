# Сакуровая коллекция для Алины

Статический сайт для GitHub Pages.

## Как добавить карты

Создай папку:

```txt
assets/cards/
```

Положи туда изображения примерно 2:3:

```txt
001_jaijijo.png
002_faofi.png
003_card.png
004_card.png
005_card.png
006_card.png
...
020_card.png
```

В `script.js` список `CARD_FILES` задаёт порядок.

## Новая механика

Каждый день на экране показываются 5 возможных подарков.

В конце выдаётся только 1 карта: первая карта текущей пятёрки.

Пример:

```txt
День 1: показываются 001–005, выдаётся 001
День 2: показываются 006–010, выдаётся 006
День 3: показываются 011–015, выдаётся 011
День 4: показываются 016–020, выдаётся 016
```

Во время открытия пятёрка отображается в перемешанном порядке, поэтому итог не выглядит очевидным.

## Таймер

Открытие доступно раз в 24 часа на конкретном устройстве. Таймер хранится в `localStorage`.

## Анимации

Сейчас в ротации 6 длинных режимов:

1. Цветение судьбы
2. Падающая звезда
3. Звёздный тоннель
4. Лунная волна
5. Кристальный рассвет
6. Сакуровый венец


## Музыка

Положи музыку сюда:

```txt
assets/music/
```

По умолчанию сайт ищет:

```txt
assets/music/sakura.mp3
```

Чтобы сделать плейлист, открой `script.js` и измени блок:

```js
const MUSIC_PLAYLIST = [
  "assets/music/sakura.mp3",
  "assets/music/song-2.mp3",
  "assets/music/song-3.mp3"
];
```

Если песня одна, она будет играть по кругу.
Если песен несколько, они будут играть по очереди.

Важно: Chrome, Safari и мобильные браузеры часто запрещают музыку со звуком до первого клика. Поэтому сайт пробует включить музыку при открытии, но если браузер заблокирует, музыка включится после нажатия на кнопку `♫` или первого клика по странице.
